# Instalação

- Instale o [npm] (https://nodejs.org/en/download/)
- Crie uma filial com o nome **desafio**
-Instale como dependências
`` `
npm install
`` `
- Rode a Aplicação
`` `
npm start
`` `
- Acesse http://localhost:888
- Todos os processos estão dentro da pasta public

# Tecnologias utilizadas

- Bootstrap
- HTML
- JavaScript
- CSS
